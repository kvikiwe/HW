# Домашка до 01 уроку: Створення змінних для інтернет-магазину

Нижче ви знайдете 20 текстових описів різних сутностей та станів, які існують у типовому інтернет-магазині. Ваше завдання: перетворити кожен текстовий опис на реальний рядок JavaScript-коду.

Що потрібно зробити:

- Хочаб для 10 з 20 наявних описів(на Ваш вибір) придумайте ідентифікатор: Самостійно підберіть логічну, зрозумілу та коротку назву англійською мовою для змінної.
- Використовуйте стиль `camelCase`.
- Оголосіть змінну: Виберіть правильне ключове слово (`let` або `const`). Подумайте, чи може значення такої змінної буде замінено в процесі роботи користувача із сайтом.
- Визначте тип даних та проініціалізуйте: Уважно прочитайте опис і вирішіть, який тип даних найкраще підходить для цієї сутності. Призначте змінній будь-яке логічне стартове значення обраного Вами типу(фейкові дані).

Приклади виконання завдання:

Опис:
«Показник того, чи вибрав користувач темну тему оформлення сайту замість стандартної світлої.»

Ваш очікуваний JavaScript-код:
```javascript
let isDarkModeEnabled = false; 
```

Опис:
«Інформація про службу підтримки, куди користувач може зателефонувати або написати.»

Ваш очікуваний JavaScript-код:
```javascript
const supportContact = {
  phone: "+380123456789",
  email: "help@shop.com",
  isOnline: true
};
```

Головна мета: Навчитися надавати змінним "промовисті" імена та СПРОБУВАТИ правильно підбирати типи даних.
